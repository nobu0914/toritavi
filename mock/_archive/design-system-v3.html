Design System v3

toritavi Calm Voyage

参考画像の「大きいラウンド」「白く柔らかい面」「深いネイビーの締め色」を、toritavi の旅程管理に合わせて再構成。 物販アプリの即売感は抑え、旅の安心感と進行管理の見やすさを前面に出す。

1. Design Principles

見た目を似せるのではなく、空気感を toritavi のプロダクト価値に変換する。

Soft Surfaces

白面は真っ白ではなく半透明寄り。 背景グラデーションと重ねて軽く浮かせる。

カードは 24px 前後の大きめ角丸。 旅程カード、検索、モーダルに統一感を作る。

Deep Navigation

主要導線はネイビーで固定。 ヘッダー、タブバー、主CTAに使い、迷いを減らす。

濃色面は情報整理エリアに限定。 タイムラインや固定ナビで特に効く。

Warm Progress

コーラルは行動喚起と注目点だけに使う。 新規作成、残件、価格相当の強調を担当。

状態色は淡い背景+濃い文字。 情報量が多い旅程でもうるさくならない。

4. Typography

Top Page

画面冒頭は大きい2行タイトル。 参考画像の強いヒーロー感を一覧ページに移植する。

セグメントは pill 形。 既存の Upcoming / Past をこの形に揃える。

Journey card は白面 + 24px 角丸 + 軽い影。 カバー画像がなくても成立する。

Journey Detail

詳細は上半分を白面、下半分を濃色タイムライン。 参考画像の order detail の読みやすさを応用。

時間列は淡色、内容は濃色内カード。 緊張感を出しつつ可読性を守る。

固定CTAはネイビー pill。 編集、進行中化、チェックインなど重要操作を統一。

Forms / Auth

入力欄は白背景より少し曇ったトーン。 柔らかい立体感を作る。

ログイン・新規登録は余白を広めに。 参考画像の安心感を保つ。

Navigation

TabBar は深いネイビーの固定パーツ。 現行方向と相性が良いので継続。

active 表現は線よりフレーム。 参考画像の立体感を残せる。
